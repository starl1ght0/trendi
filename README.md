# TrendChartsForArchivarius
Графики трендов для архиватора

## ТЗ

Отдельная страница для просмотра графиков трендов. Обыкновенный xy граф, белый фон, динамический компонент. Получает инфу запросом к серверу (развернут там же, где и фронт). `/api/trends/?x=[{dt_interval}]&y={col}&online={bool}`
Графики получаем, рисуем. Занимать всю доступную область, полностью резиновый фронт, если WS - свитчим протокол с http на ws, эндпоинт такой же. Хедер, другие панели скину позже, референсы позже. Пока просто подготовь форму и сам себе замокай сервер. Данных много, все отображать. Добавить возможность настроить график - подписи осей, масштаб, цвета линий.
